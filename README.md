# JAMLY
A simple timer app inspired by [sowon](https://github.com/tsoding/sowon) and written in C.

## Features
- Counts up after a second
- Countdowns!

## BUILD
### linux
```console
$ make
```

## USAGE
```console
$ ./jamly # counts up normally
$ ./jamly -c -H 6 -M 2 -S 1 # counts down from 6 hours 2 minutes and 1 second
```
